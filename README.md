# 💰 Loan Risk Prediction System

## 📌 Problem Statement
Financial institutions face challenges in identifying high-risk loan applicants, leading to defaults.

## 🚀 Solution
Developed an end-to-end Machine Learning system to predict loan repayment probability.

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📊 Features
- Predicts loan repayment probability
- Classifies customers as Low Risk / High Risk
- Real-time prediction using Streamlit app
